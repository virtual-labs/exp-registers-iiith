PIPO shift register
Simulation of a PIPO shift register built using D flip-flops.

Start- Starts the simulation after the inputs are properly selected. Default speed is 1x Select the speed through dropdown menu

Reset - Resets the simulation

Click on the green input bits to set their values.

This is a PIPO shift register. It has 4 Parallel(I1,I2,I3,I4) input bits. There is a clock which can be started by the user but it's cycle cannot be controlled.

There are 4 outputs
