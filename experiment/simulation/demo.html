SISO shift register
Simulation of a SISO shift register built using D flip-flops.

Start- Starts the simulation after the inputs are properly selected. Default speed is 1x Select the speed through dropdown menu

Reset - Resets the simulation

Click on the green input bits to set their values.


This is a SISO shift register. It has a Serial Input(SI) bit. There is a clock which can be started by the user but it's cycle cannot be controlled.
There is one output(SO)
